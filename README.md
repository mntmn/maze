# Building

```
git clone https://github.com/mntmn/maze
sudo apt install libsdl2-dev libsdl2-gfx-dev

./build_sdl.sh
```

# Running

```
./maze_sdl
```

# Controls

```
Space   Start game
ESC     Exit current screen
↑ or W  Walk forward
↓ or D  Walk backwards
←       Turn left
→       Turn right
A       Strafe left
D       Strafe right
```
